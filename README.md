# JavaLearn
## 项目目的：初学Java的最小demo
学习 Java 类的设计，增删改查的实现，文件io的读写，命令行的输入输出
除此之外：还可以练习 Git 的使用，idea 中 Git 的使用。

## 项目功能介绍：基于命令行的图书管理软件

首页共 4 项功能：展示书目、添加图书、删除图书、退出系统。
